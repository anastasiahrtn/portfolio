Ayo Berantas Judol: A Text-based Indonesian Gambling Site Detector Tool

May 2026

When 200,000 children in Indonesia have already been exposed to online gambling, it's no longer a cybercrime problem — it is a national crisis. Indonesia's Minister of Communication and Digital Affairs had recently reported that around 200,000 children had already been exposed to online gambling (tempo.co). Shockingly, around 80,000 of them were below the age of 10. This alarming statistic highlights the urgent need for effective measures to combat online gambling and protect vulnerable children from its harmful effects.

Ayo Berantas Judol is a web extension that I developed as a continuation of my work on the Indonesian illegal gambling site detector article. The extension is designed to help identify and block access to illegal gambling sites in Indonesia, within less than one second.

Ayo Berantas Judol is powered by a text-based machine learning classifier model that I trained using real-world Indonesian gambling site data. The model is designed to analyze the text content of websites and determine whether they are likely to be illegal gambling sites.

The flowchart above illustrates how the Ayo Berantas Judol web extension works. When a user visits a website, the extension first checks if the domain belongs to the whitelist of safe sites. If the domain is not on the whitelist, the extension then checks the text content of the website by putting it through the machine learning model.

The machine learning model used in this extension is the Logistic Regression model. To turn textual content into numerical data that the model can process, I used the Bag-of-Words vectorizer, which converts the text into a matrix of word counts. The logistic regression model then analyzes this matrix to determine the likelihood that the website is an Indonesian illegal gambling site. If the model predicts that the website is likely to be illegal, the extension will automatically redirect the user to a warning page, like shown below.

The landing page of the Ayo Berantas Judol extension serves as a warning to users that they have attempted to access an illegal gambling site. The page also provides resources for users to seek help if they or someone they know is struggling with gambling addiction. By redirecting users to this landing page, the Ayo Berantas Judol extension aims to raise awareness about the risks of online gambling and encourage responsible internet use.

Ayo Berantas Judol is a powerful tool in the fight against online gambling in Indonesia, providing users with a quick and effective way to identify and block access to illegal gambling sites. By leveraging machine learning technology, this extension can help protect vulnerable children and adults from the harmful effects of online gambling.
